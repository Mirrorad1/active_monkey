# recipe-symmetry-breaking-v0

**Status: validated (toy scale).** The project's flagship durable finding, as a reusable card.

Unsupervised emergence of latent structure from a *disembodied symbol stream* collapses
(see `boundary_notes/disembodied-stream-collapse-v0.json`). What breaks the symmetry is the
**RECIPE**: embodiment + grounding + continuous registered experience (belief never reset) +
**one** innate anchor (give the sensory map `A` *or* the motor model `B`, not both learned from
noise — Exp 31) + a few **taught** word↔concept labels.

On that recipe a toy active-inference creature runs the full chain: perceives (place fields
self-organize) → learns facts → forms grounded valence (low free energy = comfortable) →
plans + acts → **forms its own values** (same architecture + different history → different
opinion, Exp 26) → **answers in words** what it values (Exp 28/34/35).

**Honesty:** functional only, toy scale (3-color value space, small grids). The verbal answer
is a **taught template** — the *labels* are taught, the *content* (which feature it values and
why) is self-formed. Fully tabula-rasa structure and emergent compositional grammar remain
**documented research frontiers** (`open_problem.html`), not achieved. Source: Exp 26–35;
re-confirmed on the continuous-substrate creature (Exp 141–151).
