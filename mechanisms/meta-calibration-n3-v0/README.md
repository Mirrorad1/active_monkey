# meta-calibration-n3-v0

**Status: validated (at toy richness).** Agency-over-metacognition: a third-order self-model
(N3) that detects miscalibration in the creature's own metacognitive diagnoses (N2) and
rewrites N2's window parameter, passing the anti-regress test (N3 owns a regime-adaptive
control surface N2 lacks and no constant window matches).

The stack: a per-place expected-uncertainty channel + an OK/NOISE/STRUCTURAL classifier (N2
prereq, Exp 155–159); a **forecast-scoring trust monitor** (N2 labels make implicit promises,
scored against the creature's own next-100-step record — trust gap 1.0000 valid vs 0.6897
broken; a Brier alternative *fails*, tracking world difficulty not brokenness); and a
**lock-on-label-consistency controller** that climbs the window and locks on STRUCTURAL (8/8
forks, combined score 1.0 vs 0.7 for the best constant).

**Honesty:** all N3 constants (lock length K=8, dial set, trust thresholds) are **provided**.
The **ratchet law** is named explicitly: a consistency-locked dial is ascent-only (a too-large
dial in a valid world looks consistent from the inside, so the freeze holds it high — Exp 168
valid-segment deficit 0.169 > 0.15 bar). Behavioral consequences (acting on the repaired
diagnosis) are **untested** — this is diagnosis-layer authority only. Full synthesis:
`docs/research/n3-meta-calibration-chapter.md`.
