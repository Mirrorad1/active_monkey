# direction: transfer

**Question.** Does the RECIPE produce reusable structure — can a creature carry what it
learned into a NEW world, or is every world learned from zero?

**Why it matters.** Reuse is the difference between "a learning procedure" and "a mind
with knowledge." If nothing transfers, the recipe is weaker than it looks; that would be
a NEW negative finding, not consolidation.

**Experiment ladder.**
1. Same world, new geometry: creature with a learned 4x4 map dropped into a different
   4x4 layout. Does prior structure speed up (fewer steps to recovery) or actively hurt
   (interference)? FAIL = no difference vs. a fresh creature.
2. Value transfer: creature that self-formed "likes green" moved to a world where green
   appears in new places. Does it navigate to green without re-learning the value?
   FAIL = value doesn't survive the move.
3. Partial-overlap world: half familiar, half novel. Does belief stay calibrated
   (uncertain where novel, confident where familiar)? FAIL = confidence is uniform.
4. Anchor transfer: does the SAME innate anchor (motor model B) suffice in a world with
   different sensory statistics, or does the anchor encode world-specifics? FAIL =
   recipe needs a per-world anchor (this would sharpen Exp 31's claim).

**Stop condition.** All four answered (any verdict), or two consecutive experiments
blocked by the same missing substrate — then log the wall and switch direction.
