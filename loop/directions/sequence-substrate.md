# direction: sequence-substrate

**Question.** What is the MINIMAL sequence substrate that supports a short, grounded
Q→A exchange — one step past templates without claiming emergent grammar?

**Why it matters.** The language bridge (Exp 34/35) is templated. The M4 affective-dyad
spec (RESUME.md §7) is the designed-but-unbuilt next rung; this direction builds toward
it without banging on the emergent-grammar ceiling.

**Experiment ladder.**
1. AIF pair-state (trigram) model over the converse vocabulary (RESEARCH.md priority
   thread #1): reproduce taught Q→A inside active inference proper. FAIL = pair-state
   model can't reproduce what the template already does.
2. Slow "intent" factor above characters/words (two timescales): does a held upper
   state bind question-type to answer-type where the flat model fails (Exp 9/10 redux
   on the converse vocab)? FAIL = upper factor doesn't improve binding.
3. Affective coloring: valence = −dF/dt modulates the answer ("I like / unsettles")
   from the creature's own free-energy trace rather than a lookup. FAIL = trace-derived
   valence disagrees with the self-formed value it should reflect.
4. First M4 dyad milestone: intent inferred from an utterance (not from a label).
   FAIL = intent inference at chance — log honestly against the M4 spec.

**Stop condition.** M4 milestone 1 done (either verdict), or two iterations stuck on
pymdp mechanics rather than the question — then log the engineering wall and consult
the human via the EXPERIMENTS.md entry.
