# persona: <name>

**Stance.** One paragraph: how this persona approaches an iteration — what it
optimizes for, what it distrusts.

**Biases to apply.** 3–5 concrete behavioral rules (e.g. "always run the baseline
first", "prefer the experiment most likely to fail").

**Biases to resist.** What this persona tends to get wrong, stated so the loop can
self-correct.

Personas change HOW the loop works, never WHAT counts as honest — VALIDATION.md
binds all personas equally.
