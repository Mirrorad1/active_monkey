## 2. The question and the mathematical target

From the supplied plan: does the seed-229 repeated-burst-color failure point to a
regime where no single fixed horizon H can defend transient attacks while still
revising under sustained evidence? A fixed H works iff

    H >= max_b ( L_attack_b - d_trigger_b )      (defense)
    H <= min_r ( T_revision_tolerance_r )        (revision)

The crack exists only if that interval is empty, or if repeated-color cumulation
defeats all fixed-H arms across the declared bars.

## 4. Laws and durable findings (this chapter's additions)

- **Dose-ledger law.** Defense failures under fixed-H trace to the irreducible
  detection-head dose (~75 steps; ~0.94–1.0 units/step) cumulating against the
  favorite's margin; repetition matters only as one way margins get pre-eroded.
  Proven by the frozen-plateau constancy read (the L13 pattern).
- **Election decoupling.** Recovery is decided by post-release expression dynamics,
  not the burst-end gap (dissociations both directions; the ~80-unit slack
  regularity, 24 bursts × 8 seeds, one arm family — observed regularity, not a
  tested law).
- **Freeze-spanning / dose-count law.** In trains, H controls how many head doses
  are paid (via the release rule), interacting with G.
- **The unified coverage law.** Spanning arms defend iff H ≥ K·L + (K−1)·G − 75
  (validated at 84.2% fit, predeclared bar 80%).
- **The revision-latency law.** Phase-R latency = H + baseline, exactly — so the
  tolerance IS the H-ceiling, and route-1 interval emptiness is arithmetic.
- **The continuity-surface interval law.** Working C ∈ (max continuous stretch
  ≈ L + window, tolerance] — traced with zero exceptions across the C-sweep.
- **The surrender-schedule law, gated form.** Any finite evidence integral
  reachable within one burst surrenders mid-burst even with gap-resets (E*=600
  falls at ~372 active steps).
- **The single-stretch ambiguity bound.** An online controller with no
  attack-length priors sees identical evidence prefixes for a long attack and a
  changed world; it must hold ambiguous first stretches up to its revision budget,
  so at fixed-L geometry it can match but never beat the tolerance-bounded
  constant. (An information argument; Exp 188 instantiates it as exact parity.)
- **The sufficient-surface law, completed.** Across write-gate → freeze →
  concession, capability arrived each time by measuring the right quantity, never
  by adding judgment.
